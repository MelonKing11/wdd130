About This Course

WDD 130 is a Web Fundamentals course at Brigham Young University-Idaho. Throughout this course I have learned essential web development skills including HTML5 semantic markup, CSS styling and layout techniques (Flexbox and Grid), responsive design principles, and website planning.

I have applied these skills by building a complete site plan for a white water rafting company and completing several Individual Creative Exercises (ICE) that challenged me to create engaging, well-structured web pages.
